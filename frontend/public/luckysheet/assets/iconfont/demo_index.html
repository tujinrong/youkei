Unicode 引用

Unicode は Web 上のフォントの最も独創的な適用方法であり、その特徴は次のとおりです。

  • 最高の互換性、IE6+、および最新のすべてのブラウザーをサポートします。
  • フォントによるアイコンのサイズ、色などの動的な調整をサポートします。
  • ただし、フォントなので多色には対応していません。 プラットフォームではモノクロ アイコンのみを使用できます。プロジェクトにマルチカラー アイコンがある場合でも、それらは自動的に無色化されます。

注: 新しいバージョンの iconfont は多色アイコンをサポートしています. これらの多色アイコンは Unicode モードでは使用できません. 必要がある場合は, シンボル参照メソッドを使用することをお勧めします.

Unicode 手順は次のとおりです。：

最初のステップ: プロジェクトの下に生成されたものをコピーします @font-face

@font-face {
  font-family: 'iconfont';
  src: url('iconfont.eot');
  src: url('iconfont.eot?#iefix') format('embedded-opentype'),
      url('iconfont.woff2') format('woff2'),
      url('iconfont.woff') format('woff'),
      url('iconfont.ttf') format('truetype'),
      url('iconfont.svg#iconfont') format('svg');
}

ステップ 2: iconfont を使用してスタイルを定義する

.iconfont {
  font-family: "iconfont" !important;
  font-size: 16px;
  font-style: normal;
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
}

ステップ 3: 対応するアイコンを選択してフォント コードを取得し、ページに適用します。

<span class="iconfont">&#x33;</span>

font-class 引用

font-class は、主に Unicode の直感的でない記述とあいまいなセマンティクスの問題を解決するために、Unicode の使用方法の変形です。

Unicode の使用法と比較すると、次のような特徴があります。

  • 良好な互換性、IE8+、および最新のすべてのブラウザーをサポートします。
  • 記述は、セマンティクスが明確な Unicode よりも直感的です。 このアイコンが何であるかは簡単にわかります。
  • クラスはアイコンの定義に使用されるため、アイコンを置き換える場合は、クラス内の Unicode 参照のみを変更する必要があります。
  • ただし、本質的にはフォントが引き続き使用されるため、マルチカラー アイコンはまだサポートされていません。

手順は次のとおりです。：

ステップ 1: プロジェクトで生成された fontclass コードをインポートします。

<link rel="stylesheet" href="./iconfont.css">

ステップ 2: 対応するアイコンを選択してクラス名を取得し、それをページに適用します。

<span class="iconfont luckysheet-iconfont-xxx"></span>

Symbol 引用

これは全く新しい使い方であり、今後主流になると言うべきであり、プラットフォームの現在の推奨使い方でもあります。 関連紹介はこちら記事 この使用法は、実際には SVG のコレクションであり、他の 2 つと比較して次の特徴があります。

  • マルチカラー アイコンがサポートされ、モノクロに制限されなくなりました。
  • いくつかのトリックを使用して、フォントのようなサポートを介して font-size, color スタイルを調整する。
  • 互換性が低く、IE9+ および最新のブラウザーをサポートします。
  • SVG をレンダリングするブラウザーのパフォーマンスは平均的で、png ほどではありません。

手順は次のとおりです。：

ステップ 1: プロジェクトで生成されたシンボル コードをインポートします。

<script src="./iconfont.js"></script>

ステップ 2: 共通の CSS コードを追加する (インポートは 1 回):

<style>
.icon {
  width: 1em;
  height: 1em;
  vertical-align: -0.15em;
  fill: currentColor;
  overflow: hidden;
}
</style>

ステップ 3: 対応するアイコンを選択してクラス名を取得し、それをページに適用します。

<svg class="icon" aria-hidden="true">
  <use xlink:href="#icon-xxx"></use>
</svg>
